STEP.4 모듈 미사용 데이터 호출

            

                
    
                    
  1. 상단 모듈을 사용하지 않는 디자인 영역에서의 데이터를 호출하는 경우는 하단의 코드를 참고하여 클래스를 추가할 수 있습니다. 
    2. 
                

            

        

        

           
기업 기본정보

           

                기업의 기본정보는 기업명, 사업자등록번호, 기업형태, 업종, 대표자명, 전화번호, 우편번호 및 주소, 홈페이지, 기업로고의 정보를 데이터로 출력합니다.

                기업명 : .comName / 사업자등록번호 : .comNum /  기업형태 : .comForm /
                업종 : .incCate / 대표자명 : .ceo / 전화번호 : .tel / 
                전체주소 : .address / 주소만 : .comAddress / 기업로고 : .imgCi /
                기업대표이미지 : .imgCom / 홈페이지 : .site / 홈페이지 바로가기 : .siteGo
